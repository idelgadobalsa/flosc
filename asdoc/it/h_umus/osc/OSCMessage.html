Packageit.h_umus.osc
Classpublic class OSCMessage

An OSCMessage object is the data unit for encapsulating OSC messages. OSC messages, as in OSC specs, are defined by a mandatory name and a following collection of OSC arguments.

See also

http://code.google.com/p/flosc

public function getXML():XML

This methods generates and XML rapresentation of the OSCMessage defined by the Flosc DTD.

It would look like : 

   
     <MESSAGE NAME="/osc/address">
      <ARGUMENT TYPE="i" VALUE="2"/>
      <ARGUMENT TYPE="f" VALUE="3.0"/>
     </MESSAGE>
