Packageit.h_umus.osc
Classpublic class OSCArgument

An OSCArgument object is the data unit for encapsulating OSC arguments. An OSCArgument is composed by an osc argument type defined in the OSC specs and it's value. As some limits may appear during this implementation and the Flosc server implementation not all osc types and values are supported at the moment of this writing. Future updates on this libary and the Flosc server may upgrade for new types and values.
